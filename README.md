## Installation & Setup

### Step 1
```
$ git clone https://github.com/AzureSource/GoalSeekr.git
$ cd GoalSeekr
$ npm install
$ npm run build
$ npm run server-dev
```
### Step 2
- Create a .env file based on example.env
- Copy and Paste everything from example.env to .env
- replace user details with your own
### Step 3
- Open http://localhost:7777
